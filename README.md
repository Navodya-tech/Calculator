# AWT Calculator

This is a simple Java desktop calculator built using **AWT (Abstract Window Toolkit)**. It supports **basic arithmetic operations** like addition, subtraction, multiplication, and division, as well as **square root** and **percentage calculations**. The calculator also includes **clear (CE) and backspace (X) functionality** for easy input correction. The project demonstrates **GUI creation, event handling, and layout management** in Java, making it a great example for learning Java desktop application development.
